# Ntfy Monitor Extension

This Chrome extension monitors a user-specified ntfy.sh channel and provides notifications for new entries. It displays browser toasts for new messages and allows users to view all notifications through the extension's popup interface. The extension also indicates unread messages on its icon.

## Features

- Monitors a specified ntfy.sh channel for new notifications.
- Displays toast notifications for new entries.
- Shows all notifications in a user-friendly popup interface.
- Indicates unread messages on the extension icon.

## Installation

1. Download or clone the repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click on "Load unpacked" and select the `ntfy-monitor-extension` directory.
5. The extension should now be installed and visible in your extensions list.

## Usage

1. Click on the extension icon in the Chrome toolbar to open the popup.
2. Enter your ntfy.sh channel in the designated input field.
3. The extension will start monitoring the channel and display notifications for new entries.
4. Click the extension icon again to view all notifications.
